## ✨ 功能

🎯 **智能消息助手**：自动发送消息，告别重复机械喊话

📦 **商品自动讲解**：自动商品弹窗，随心所欲弹讲解

🤖 **AI 智能助理**：接入 DeepSeek，支持官方 api 以及 openrouter api
